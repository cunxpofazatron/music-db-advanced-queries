# Музыкальная база данных

Домашнее задание к лекции «Продвинутая выборка данных».

## Структура проекта
- create_tables.sql — создание таблиц
- insert_data.sql — заполнение базы данных
- select_queries.sql — SELECT-запросы (задания 2–4)

## Описание
База данных музыкального сервиса:
- Исполнители
- Жанры
- Альбомы
- Треки
- Сборники

Реализованы связи many-to-many и сложные SQL-запросы с JOIN, GROUP BY, подзапросами.
